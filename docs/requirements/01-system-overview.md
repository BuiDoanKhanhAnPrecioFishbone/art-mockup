# 01 — System Overview

> Index → see [INDEX.md](./INDEX.md). End-to-end recap → [11-end-to-end-flow.md](./11-end-to-end-flow.md).

Precio Fishbone is an end-to-end recruitment + assessment platform. It
manages the full hiring lifecycle (job templates → program setup →
candidate intake → review → testing → interviewing → final decision)
plus a standalone Assessment Management module for non-recruitment skills
testing.

## Platform modules

| Module | Purpose | Primary users |
| --- | --- | --- |
| **Recruitment & Program** | Manage hiring programs, candidates, workflows, stages, steps. | HR, Recruiters |
| **Assessment Management** | Build and manage the question bank, tests, and test sessions. | HR, Reviewers |
| **Submission (Sessions)** | Create and oversee time-bounded test sessions; track submissions and integrity. | HR, Reviewers |
| **Template Library** | Maintain reusable email, skill, flow, and job templates. | HR Admin |
| **Administration** | Manage users, roles, and metadata. | System Admin |

## Sidebar navigation (4 groups)

- **Assessment Management** — Question, Test, Submission
- **Recruitment & Program** — Programs, Candidates, Reports
- **Template Library** — Email Template, Skill Template, Recruitment Flow
  Template, Job Template (and our addition: Interview Criteria Template)
- **Administration** — Manage Users, Manage Metadata
